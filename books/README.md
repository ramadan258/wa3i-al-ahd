# مكتبة التعافي

لإضافة كتاب جديد يظهر لجميع أعضاء الموقع:

1. ضع ملف الكتاب داخل هذا المجلد.
2. افتح الملف `index.html`.
3. ابحث عن `RECOVERY_BOOKS`.
4. أضف عنصرًا جديدًا بهذا الشكل:

```js
{
  id: "book-1",
  title: "اسم الكتاب",
  author: "اسم المؤلف",
  description: "وصف قصير أو فائدة الكتاب",
  href: "books/اسم-الملف.pdf",
  minutes: 5
}
```

ملاحظات:

- `href` يمكن أن يكون ملفًا داخل المجلد `books/` أو رابطًا عامًا مباشرًا.
- `minutes` اختيارية، وهي فقط لعرض وقت قراءة مقترح.
- بعد الإضافة سيظهر الكتاب داخل `مكتبة التعافي` لجميع الأعضاء.
